## Тестовое задание React

Нужно разработать приложение со списком todo в формате Single Page Application.

### Требования к приложению.

Приложение должно содержать 2 страницы: главная и список todo
Переход между страницами должен быть реализован через navbar
На главной странице должно быть лого и краткое описание страницы
На странице для списка todo должен быть интерактивный список
Данные для списка должны быть запрошены с https://jsonplaceholder.typicode.com/todos
Должна быть реализована возможность добавления/изменения/удаления todo
Должна быть реализована пагинация без перезагрузки страницы

### Нужно использовать:

React
Redux
React router

### Будет плюсом:

Next js
Typescript

### Можно использовать:

Axios
Css библиотеки

### Нельзя использовать:

Библиотеки с готовыми компонентами или плагины для React, которые предоставляют готовый функционал, требуемый в задании CMS системы

### Критерии оценки:

Работоспособность согласно ТЗ
Архитектура решения
Качество кода
Удобство проверки
